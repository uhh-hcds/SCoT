Infos on heuristic The neighbourhood of a node is balanced, if there are at least two clusters in the node's neighbourhood for which
max - d_i < mean / 2
holds true.
  • max: the highest number of NGOT undirected links between the node and the nodes belonging to one cluster
  • d_i: the number of NGOT undirected links between this node and the nodes belonging to cluster i
  • mean: the mean number of NGOT undirected links between this node and the nodes of any of all clusters
